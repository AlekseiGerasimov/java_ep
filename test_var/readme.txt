Папка DB:
Я выбрал вариант с базой данной, выбрана база из рекомендованных (SQLite).В данной папке расположена физическая схема БД, запросы для создания таблиц, заполнения таблиц и запрос по всем дефектам от конкретного пользователя по конкретному проекту
Папка java_code:
Выбрана следующая структура проекта: при заходе в главное меню проекта есть возможность работать с 6 сущностями:
1. Проекты (project) - при выборе данного пункта меню можно работать с проектами (добавлять, удалять, редактировать). У сущностей Пользователи (users) и Дефекты (issues) аналогичный функционал.
2. Загрузка данных из файла в базу данных - при выборе данного пункта в базу можно загрузить проекты, пользователей или дефекты из файла.
3. Отчеты - можно вывести все дефекты от конкретного пользователя по конкретному проекту и вывести все дефекты по конкретному проекту
4. Прикрепить пользователя к проекту - так как таблицы пользователи и проекты связаны отношением многие ко многим , то нужна была таблица для связывания пользователя с проектом, данный пункт меню привязывает существующего пользователя к существующему проекту.

Все исключения я постарался обработать с помощью библиотеки java.util.logging, все ошибки пишутся в файл logs.txt.
